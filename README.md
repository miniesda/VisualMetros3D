Metro de sevilla augmented reality in Next.js typescript and
Virtual representation of Sevilla, Mayorca and Bilbao metro systems in Unity

using standard installation of next.js by Official Documentation

dependency:

```
cd my-app
nvm install 20.9.0
nvm use 20.9.0
npm install
npm run dev
```

Open [http://localhost:3000/VisualMetros3D](http://localhost:3000/VisualMetros3D) with your browser to see the result.

IMPORTANT!! dont use node ^v21 as npm canvas didnt deploy package yet If node 21 or higher is needed, use the javascript release

extra dependencies included:

```
npm install aframe
npm install aframe-react
npm install mind-ar
```
